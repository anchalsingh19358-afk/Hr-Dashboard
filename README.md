# Hr-Dashboard
### 1. Project Title / Headline
A cocise, description name for the dashboard. 
Example:
💼 HR Analytics Dashboard: Employee Insights & Attrition Analysis
An interactive and data-driven Power BI dashboard designed to analyze workforce trends, employee attrition, job satisfaction, and demographic patterns—helping organizations make smarter HR decisions.

### 2. Short Description / Purpose
1-2 sentences explaning what the dashboard does and why it exists.
The HR Analytics Dashboard is a comprehensive Power BI report that provides insights into employee distribution, attrition trends, age demographics, and job satisfaction. It helps HR teams and business leaders understand workforce behavior and take data-driven actions to improve retention and performance.

### 3. Tech Stack
The dashboard was built using the following tools and technologies:
List the key technologies used to build the dashboard.
• 📊 Power BI Desktop – Main platform used for building interactive dashboards
• 📂 Power Query – Used for data cleaning, transformation, and shaping
• 🧠 DAX (Data Analysis Expressions) – Used for KPIs, calculated measures, and dynamic insights
• 📝 Data Modeling – Relationships created between employee data tables for accurate analysis
• 📁 File Formats – .pbix for development and .png for preview

### 4. Data Source
Source: HR dataset (employee records)
The dataset includes detailed employee information such as:
Employee count and attrition
Department (HR, Sales, R&D)
Age groups and gender
Education fields
Job roles and satisfaction ratings
This structured dataset enables multi-dimensional analysis of workforce trends and HR performance.

### 5. Features / Highlights
• Business Problem
Organizations often struggle to understand why employees leave, which departments are most affected, and how factors like age, job role, and satisfaction influence attrition.

### Key questions include:
Which department has the highest attrition?
What age group is most likely to leave?
How does job satisfaction impact retention?
Which roles are most affected?

• Goal of the Dashboard
To create an interactive HR analytics tool that:
Tracks employee attrition and workforce distribution
Identifies high-risk groups and departments
Analyzes job satisfaction trends
Supports strategic HR decision-making

• Walkthrough of Key Visuals
- Key KPIs (Top Section)
Total Employees: 1470
Attrition Count: 237
Attrition Rate: 16.12%
Active Employees: 1233
Average Age: 37
- Department-wise Attrition (Pie Chart)
  R&D, Sales, and HR attrition comparison
  Highlights which department faces the highest employee turnover.
- Employee Distribution by Age Group (Bar Chart)
  Age groups: Under 25, 25–34, 35–44, 45–54, Over 55
  Shows workforce concentration across different age segments.
- Job Satisfaction Ratings (Table)
  Satisfaction levels (1–4) across job roles
  Helps identify roles with low satisfaction, which may lead to attrition.
- Education-wise Attrition (Bar Chart)
  Fields like Life Sciences, Medical, Marketing, etc.
  Reveals which educational backgrounds have higher attrition.
- Age-wise Attrition by Gender (Donut Charts)
  Gender comparison across age groups
  Identifies demographic patterns in employee exits.

### • Business Impact & Insights
✔ Employee Retention Strategy
HR teams can identify high-attrition departments and take targeted action.

✔ Workforce Planning
Understand which age groups dominate the workforce and plan hiring accordingly.

✔ Improved Job Satisfaction
Detect roles with low satisfaction and improve work conditions.

✔ Data-Driven Decisions
Management can use insights to reduce attrition costs and improve productivity.

✔ Strategic HR Insights
Helps align HR policies with real workforce trends and behaviors.
